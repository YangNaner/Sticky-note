# 键盘快捷键与便签缩放设计

## 概述

为便利贴应用添加键盘快捷键和鼠标滚轮缩放功能，提升操作效率。

## 功能设计

### 1. 键盘快捷键

| 快捷键 | 绑定位置 | 功能 |
|--------|----------|------|
| `Ctrl+N` | app (全局) | 新建便签 |
| `Ctrl+T` | 当前便签 | 切换置顶状态 |
| `Esc` | 当前便签 | 销毁当前便签 |

**实现细节**
- `Ctrl+N` 在 app 层面绑定，新建便签逻辑复用现有 `add_note_to_main()`
- `Ctrl+T` 在 StickyNote 类绑定，调用现有 `toggle_topmost()` 方法
- `Esc` 在 StickyNote 类绑定，调用现有 `destroy_note()` 方法

### 2. 鼠标滚轮缩放

- 事件绑定：鼠标滚轮滚动时触发
- 缩放目标：当前焦点便签
- 缩放方式：调整窗口高度
- 高度变化：每次 ±30px
- 范围限制：最小 150px，最大 600px
- 字体大小：保持 11px 不变

**焦点管理**
- 点击便签任意区域时设置为活动便签
- 滚轮事件只作用于活动便签
- 使用实例变量 `_focused` 跟踪焦点状态

## 代码改动

### main.py
- 无改动

### sticky_note.py
1. 添加 `_focused` 实例变量
2. `__init__` 中绑定 `Ctrl+T` 和 `Esc` 快捷键
3. 点击便签时设置 `_focused = True`
4. 添加 `on_mousewheel` 方法处理缩放
5. 绑定鼠标滚轮事件

## 测试场景

1. `Ctrl+N` 全局新建便签
2. `Ctrl+T` 切换置顶状态，菜单文字更新
3. `Esc` 销毁当前便签
4. 滚轮上滑便签变大，下滑变小
5. 边界检查：最小/最大高度限制生效
